# discord.py-builder-Extensions
discord.py-builderの拡張機能です。

# discord.py-builder-Extensionsとは？
discord.py-builderにブロックやカテゴリを追加する、拡張機能です。<br>

# 何が追加されるの？
1. VC関連のブロック
2. DM関連のブロック
3. 招待リンク関連のブロック
4. Webリクエスト関連のブロック
5. Break、returnなどのブロック
6. OSコマンド実行用ブロック
7. ほかにもいっぱい！

# インストールするには？
このレポジトリをzipでダウンロードし、<br>
解凍後、Chromeに読み込ませます。<br>
最後にdiscord.py-builderを開いて反映されていればOK!
